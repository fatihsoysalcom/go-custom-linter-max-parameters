# Go Custom Linter Max Parameters

This Go program demonstrates how to create a basic custom linter. It parses its own source code into an Abstract Syntax Tree (AST) and then traverses it to find functions that exceed a predefined maximum number of parameters. Issues are reported with file, line, and column information.

## Language

`go`

## How to Run

Save the code as `main.go`.
Run from your terminal: `go run main.go`
